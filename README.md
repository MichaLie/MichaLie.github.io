# michalie.github.io — AI for Researchers

Landing page for a curated collection of open, researcher-facing AI indexes, **dedicated to [ELIXIR-CZ](https://www.elixir-czech.cz/)** (the Czech national node of [ELIXIR](https://elixir-europe.org/)).

**▶ Live:** https://michalie.github.io/

The collection:
- **[Foundation Models](https://michalie.github.io/bio-foundation-models-wiki/)** — pretrained models for DNA, RNA, proteins, structures, molecules, single-cell omics, and pathology.
- **[Autonomous Science Agents](https://michalie.github.io/autonomous-stem-agents-wiki/)** — AI agents, self-driving labs, and benchmarks across STEM.
- **[Coding & Data Agents](https://michalie.github.io/research-coding-agents-wiki/)** — privacy-first AI coding & data-analysis tools, matched to your data sensitivity.

A self-contained static `index.html`, served by GitHub Pages. Each index is a discovery aid — verify details with the source before relying on them.

Created by Michaela Liegertová.
